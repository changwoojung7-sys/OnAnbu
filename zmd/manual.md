# 📖 ON-ANBU (온안부) 사용자 메뉴얼

> **"부모님과 자녀를 연결하는 따뜻한 안부 서비스"**
>
> 🌐 웹 서비스: [https://onanbu.calamus.ai.kr](https://onanbu.calamus.ai.kr)

---

## 1. 시작하기

### 1-1. 회원가입

#### 👩‍👧 자녀(주 보호자) 가입

1. 앱 또는 웹 접속 → **회원가입** 클릭
2. 이름, 이메일, 비밀번호 입력
3. 이메일로 전송된 **OTP 인증코드** 입력
4. 인증 완료 → 자동 로그인

#### 👴 부모님 가입

1. 자녀가 먼저 **부모님 초대 코드**를 생성 (아래 2-4 참조)
2. 앱 접속 → **초대코드로 시작하기** 클릭
3. 초대코드 입력 → 이름, 간편 비밀번호 설정
4. 즉시 로그인 → 부모님 전용 화면 이동

#### 👤 보조 보호자 가입

1. 주 보호자가 **보조 보호자 초대 코드**를 생성 (아래 2-5 참조)
2. 앱 접속 → **초대코드로 시작하기** 클릭
3. 초대코드 입력 → 초대한 부모님 정보 확인
4. 이메일, 비밀번호 설정 → 즉시 로그인

### 1-2. 로그인

- 이메일 + 비밀번호 입력 → **로그인** 클릭
- 비밀번호 분실 시: **비밀번호 찾기** → 이메일로 OTP 수신 → 새 비밀번호 설정

## 2. 자녀(보호자) 화면 가이드

### 2-1. 🏠 홈 탭

- 연결된 부모님의 **오늘 안부 상태**를 한눈에 확인
- 부모님이 아직 안부를 보내지 않았으면: "아직 아침 소식이 없어요"
- 부모님이 안부를 보냈으면: 최근 활동 카드 표시 (사진, 메시지 등)
- **여러 부모님** 등록 시: 상단 "다른 부모님 선택 ▼"으로 전환 가능

### 2-2. 💚 케어 탭 (마음 전하기)

부모님께 사진, 영상, 음성으로 마음을 전할 수 있어요.

#### 미디어 응원 전하기

1. **"광고 보고 선물 보내기"** 버튼 클릭
2. 미디어 선택 모달에서:
   - 📷 **사진**: 갤러리에서 사진 선택
   - 🎬 **동영상**: 갤러리에서 영상 선택
   - 🎙️ **음성**: 직접 녹음 (녹음 버튼 → 정지 버튼)
3. 미디어 첨부 확인 → **"광고 보고 전하기"** 클릭
4. 15초 광고 시청 (부모님께는 광고가 보이지 않아요)
5. 전송 완료! 💌

#### 오늘 안부 대신 기록하기

1. **"15초 광고로 완료하기"** 클릭
2. 부모님 대신 텍스트 안부 작성
3. 광고 시청 후 전송

### 2-3. 📋 기록 탭 (히스토리)

- 부모님과 자녀 간의 **모든 활동 기록**을 타임라인으로 확인
- 사진/영상은 카드 형태로 표시, 클릭하면 원본 보기
- 음성 메시지는 ▶️ 재생 버튼으로 바로 듣기
- 기분 이모지 배지로 부모님의 하루 컨디션 확인 가능
- **"우리가족 전체 기록 보기"**로 전체 히스토리 이동

### 2-4. 👨‍👩‍👧 부모님 초대하기

1. 홈 탭 → **가족 관리** 또는 설정에서 **부모님 초대**
2. 6자리 초대코드 생성됨
3. 코드를 부모님께 알려주세요 (카카오톡, 문자 등)
4. 부모님이 코드로 가입하면 자동 연결!

### 2-5. 👤 보조 보호자 초대하기

1. 가족 관리 → **보조 보호자 초대**
2. 케어할 부모님 선택
3. 초대코드 생성 → 형제/자매에게 공유
4. 보조 보호자가 코드로 가입하면 같은 부모님의 안부를 함께 확인 가능

## 3. 부모님 화면 가이드

### 3-1. 🌅 홈 화면

큰 버튼으로 간편한 안부 전송!

#### 기상 알림

- **"일어났어요! ☀️"** 버튼 터치 (1일 1회)
- 자녀에게 기상 알림이 전달됩니다

#### 기분 체크

- 😊😐😢😡🤒 이모지 중 오늘의 기분 선택
- 자녀가 히스토리에서 확인할 수 있어요

#### 미디어 전송

- **📸 사진/영상**: 갤러리에서 선택하여 전송
- **🎙️ 음성 메시지**: 목소리로 안부 전송 (녹음 → 전송)

### 3-2. 📬 자녀가 보낸 안부 확인

- 홈 화면 하단에 자녀가 보낸 최근 안부가 표시됩니다
- 사진, 영상, 음성 메시지를 바로 재생/확인 가능
- 발신자 이름과 프로필이 함께 표시됩니다

### 3-3. 📜 전체 기록 보기

- **"우리가족 전체 기록 보기"** 버튼으로 전체 히스토리 확인
- 날짜별로 정렬된 활동 기록을 스크롤하며 확인

### 3-4. ⚙️ 설정

- **프로필 수정**: 이름, 프로필 사진 변경
- **알림 설정**: 알림 받기 ON/OFF, 알림 시간 설정
- **로그아웃**

## 5. 자주 묻는 질문 (FAQ)

### Q. 부모님이 스마트폰에 익숙하지 않아요
>
> 부모님 화면은 **큰 버튼과 간단한 UI**로 구성되어 있어요. "일어났어요" 버튼 하나만 눌러도 안부가 전달됩니다.

### Q. 광고를 꼭 봐야 하나요?
>
> 자녀가 미디어를 전송할 때 15초 광고를 시청합니다. **부모님께는 광고가 전혀 보이지 않아요.** 광고 시청은 "효도 선물 포장"과 같은 개념입니다.

### Q. 여러 자녀가 한 부모님을 케어할 수 있나요?
>
> 네! 주 보호자가 **보조 보호자 초대코드**를 생성하여 형제/자매를 초대하면, 모두 같은 부모님의 안부를 확인하고 소통할 수 있습니다.

### Q. 웹과 앱 모두 사용할 수 있나요?
>
> 네! 웹 브라우저([onanbu.calamus.ai.kr](https://onanbu.calamus.ai.kr))에서 바로 사용 가능하고, 안드로이드 앱도 지원합니다.

### Q. 보낸 기록을 삭제할 수 있나요?
>
> 본인이 보낸 기록은 히스토리에서 🗑️ 삭제 버튼으로 삭제할 수 있습니다.
